## 📊 技术规格

### 模型架构
- **参数量：** 6B（高效快速）
- **架构：** S3-DiT（可扩展单流扩散Transformer）
- **推理步数：** 8步（Turbo）/ 50步（Base）
- **训练：** 从Z-Image Base模型蒸馏而来

### 支持功能
- **图片尺寸：** 1:1、16:9、9:16、4:3、3:4
- **最大分辨率：** 2048×2048
- **输出格式：** PNG
- **文字渲染：** 出色的中英文文字生成
- **提示词语言：** 中文、英文或混合

### 性能表现
- **H100 GPU：** < 1秒
- **消费级GPU（16GB）：** 5-10秒
- **质量：** 照片级真实，媲美顶级商业模型
- **速度：** 比Flux.1快4倍，比DALL-E快2-3倍
